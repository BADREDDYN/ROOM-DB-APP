# ROOM-DB-APP
Simple Room database app.

> Add the dependencies:

  ```gradle
  plugins {
    ...
    id 'kotlin-kapt'
  }
  ...
	dependencies {
    ...
    implementation 'androidx.room:room-runtime:2.4.3'
    kapt 'androidx.room:room-compiler:2.4.3'
    implementation 'androidx.room:room-ktx:2.4.3'
	}
  ```
